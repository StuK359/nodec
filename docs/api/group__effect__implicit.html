Detailed Description

Functions to easily define and use implicit parameters.

Example

implicit_define(width);
void bar() {
printf("width is %i\n", lh_int_value(implicit_get(width)));
}
void foo() {
{using_implicit(lh_value_int(42),width){
bar();
}}
}
