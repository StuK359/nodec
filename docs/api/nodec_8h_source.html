17 /* ----------------------------------------------------------------------------
18  Notes:
19  - Private functions are prepended with an underscore. Dont use them directly.
20  - Pre- and post-fixes:
21  "async_" : for asynchronous functions that might interleave
22  "nodec_" : synchronous functions that might throw exeptions or use other effects.
23  "nodecx_": synchronous functions that return an explicit error result.
24  "..._t" : for types
25  "using_" : for scoped combinators, to be used with double curly braces:
26  i.e. "{using_alloc(tp,name){ ... <use name> ... }}"
27 
28  - an `lh_value` is used to mimic polymorphism. There are a host of
29  conversion functions, like `lh_ptr_value` (from value to pointer)
30  or `lh_value_int` (from int to a value).
31 -----------------------------------------------------------------------------*/
