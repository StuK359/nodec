Detailed Description

Implicit bindings for HTTP incoming- and outgoing data for a server.

The HTTP(S) server implicitly binds the current request and response objects (as dynamically bound implicit parameters). They can be accessed through http_req_xxx for requests and http_resp_xxx for responses.
